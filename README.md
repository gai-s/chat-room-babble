# Babble - Real-time Chat App

Babble is a real-time chat application that allows users to communicate instantly in private or group chats. 
MERN application, Web-based responsive chat room, retro UI style.

## Project Goal
Demonstrates frontend skills with React and real-time communication using WebSockets.

## Key Features
- Real-time messaging between users
- Active user list display
- Clean and intuitive UI

## Technologies
React, Redux, Node.js, Express, Socket.io, JWT, md-5 & Gravatar API, CSS Modules
Unit testing: Jest

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
